Guía para realizar el Ejercicio 2

Curso: Programación orientada a objetos - Java avanzado

🎯 Objetivo

Ampliar la aplicación web para que cada Cliente pueda tener una lista de Pedidos. Se implementarán nuevas vistas, controladores y validaciones.

🛠️ Paso 1: Ampliación del modelo de dominio

  • Crear clase Pedido con atributos articulo y precio.
  • Modificar clase Cliente:
    • Añadir ArrayList<Pedido> pedidos.
    • Eliminar numPedidos como atributo fijo.
    • Implementar getNumPedidos() como método calculado.

🗃️ Paso 2: Adaptación del DAO

  • Implementar patrón Singleton en ClienteDao.
  • Usar getInstance() para obtener la instancia única desde los controladores.

👤 Paso 3: Creación y actualización de clientes

  • Eliminar campo numPedidos del formulario cliente.jsp.
  • Modificar ClienteController para no sobrescribir el cliente completo al actualizar.

📋 Paso 4: Modificación de la vista inicial

  • En listadoClientes.jsp, añadir enlace “Pedidos” que apunte a PedidoController?action=listado&clienteId=X.
  • La columna de número de pedidos seguirá funcionando gracias a getNumPedidos().

📦 Paso 5: Listado de pedidos

  • Crear PedidoController y vista listadoPedidos.jsp.
  • Mostrar datos del cliente y sus pedidos en una tabla con <c:forEach>.
  • Añadir enlace para insertar pedido.

📝 Paso 6: Creación de pedidos

  • En PedidoController, manejar action=insertar en doGet.
  • Crear pedido.jsp con formulario para articulo y precio.
  • En doPost, añadir pedido al cliente y redirigir.

✅ Paso 7: Pruebas

  • Insertar varios clientes y pedidos.
  • Verificar que los pedidos no se mezclan.
  • Comprobar que al actualizar un cliente no se pierden sus pedidos.

🔒 Paso 8: Validaciones (Opcional)

  • Validar campos obligatorios, formato de fecha y precio.
  • Se recomienda usar Bean Validation (Hibernate Validator).
  • Mostrar mensajes de error en pantalla.

Para más información sobre Bean Validation, consulta la documentación oficial de Jakarta Bean Validation API.

📦 Entrega: Código fuente del proyecto (Eclipse) en un archivo ZIP.
