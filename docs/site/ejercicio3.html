Introducción

Se parte del servicio web del Tema 5 que permite obtener, crear y editar ciudadanos. El objetivo es ampliar el API con un método de borrado, modificando dos proyectos Eclipse: Servidor y Cliente.

Primera Parte: Servidor

Paso 1: Modificación del DAO

  • Clase: CiudadanosDAO
  • Método: deleteCiudadano(int id)
  • Características:
    • Argumento: ID del ciudadano
    • No devuelve nada
    • Borra el ciudadano si existe

Paso 2: Modificación del API

  • Clase: CiudadanoAPI
  • Método: delete con anotaciones @Path, @PathParam, @DELETE
  • URL: /api/ciudadano/{id} usando método HTTP DELETE

Paso 3: Prueba con SoapUI

  • Crear nuevo proyecto REST
  • Importar nuevo WADL
  • Realizar peticiones DELETE y GET
  • Capturar pantallas para entrega

Segunda Parte: Cliente

Paso 4: Modificación del API

  • Clase: ConsumidorCiudadanoAPI
  • Método: delete(int id)
  • Devuelve: true si la operación es correcta
  • Uso de webTarget y invocation builder

Paso 5: Prueba del Cliente

  • Leer ciudadanos y mostrarlos
  • Borrar el primero si la lista no está vacía
  • Volver a leer y mostrar ciudadanos

Entrega del Ejercicio

  • ZIP del proyecto Eclipse del Servidor
  • ZIP del proyecto Eclipse del Cliente
  • Documento con capturas de SoapUI
  • Entrega completa, no clases sueltas
